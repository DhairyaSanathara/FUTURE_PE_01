## Prompt Workflow (Step-by-Step)

### Step 1 — Brand Strategy (Prompt 1)
A structured brand profile is created including:
- Brand personality
- Customer personas
- Customer problems
- Unique Value Proposition (UVP)
- Voice & tone guidelines
- Key messaging themes

### Step 2 — Homepage Generation (Prompt 2)
Generates homepage copy including:
- Hero headline + subheadline
- Intro
- Why choose us
- Sip vs Zip section
- Testimonials

### Step 3 — Services Page (Prompt 3)
Generates service page copy with:
- Service hooks
- What’s included
- Benefits
- Micro CTAs

### Step 4 — Website CTAs (Prompt 4)
Generates CTA blocks for:
- Visit now
- Call/WhatsApp
- Get directions
- Offers & weekend specials
- Work mode (Wi-Fi/Seating)
- Reviews / feedback

### Step 5 — Tone Variations (Prompt 5)
Rewrites hero + intro into:
- Friendly (Gen Z vibe)
- Professional (clean minimal)
- Premium (elevated)
