# Task 1 – AI Website Copy Generator for a Local Café

## Business Scenario

Business Name: Sip & Zip  
Location: Ahmedabad, India  
Type: Modern urban fast-casual café  
Target Audience: Gen Z, digital nomads, coffee-date couples  
Core Offerings: Patience-steeped cold brews and high-speed espresso snacks  
Brand Tone: High-energy “Zip” meets low-stress “Sip”  
Unique Angle: Dual-speed service – slow workspace vs turbo pickup  

## Objective

To design a prompt-based system that generates complete, conversion-focused
website copy for a modern local café using AI tools.

## Tools Used
ChatGPT, GitHub
